# ingredients-predictor
Project made for Advanced Artificial Neural Networks at University of Warsaw

## App setup
1. Create a Python environment. Install packages from ```requirements.txt``` file
2. Upload ```final_recipes_dataset.csv``` to project directory
3. Download ```cc.en.300.bin.gz``` file from https://www.kaggle.com/datasets/sanyatargrenkin/cc-en-300-bin and unpack it in the directory
4. Run entire ```gensim_model.ipynb``` notebook and wait until the ```recipe_fasttext_mlp.pt``` file is created
5. Run the app using ```python app.py```
